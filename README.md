# Postcard Syntax Analyzer

Сервис для анализа синтаксической сложности русскоязычных текстов (открыток).  
Вычисляет 44 синтаксических параметра с помощью библиотеки spaCy и предоставляет веб-интерфейс.

## Функциональность

- Расчет глубины синтаксического дерева для каждого предложения  
- Определение расстояния между главным и зависимым словом  
- Подсчет клауз  
- Расчет длины цепочек прилагательных  
- Выявление обстоятельственных, определительных и изъяснительных придаточных  
- Подсчет причастий, деепричастий, инфинитивных оборотов, предложных определений  
- Два режима вывода:  
  – упрощенный (11 ключевых показателей)  
  – полный (все 44 параметра с русскими названиями)  
- Веб-интерфейс для интерактивного анализа
- Пакетный анализ данных из CSV-файла

## Установка и использование

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/Marina-Mokina/postcard-syntax-analysis.git
   cd postcard-syntax-analysis
   ```

2. **Установите зависимости через Poetry:**
   ```bash
   poetry install
   ```

3. **Установите языковую модель spaCy для русского языка:**
   ```bash
   poetry run python -m spacy download ru_core_news_lg
   ```

4. **Запустите веб-сервис:**
   - **С помощью скрипта:**  
     ```bash
     poetry run postcard-api
     ```
   - **Или напрямую через uvicorn:**  
     ```bash
     poetry run uvicorn backend.service:app --reload
     ```

   Сервис станет доступен по адресу `http://localhost:8000`.

5. **Пакетный анализ данных:**
   ```bash
   poetry run python backend/run_analysis.py
   ```

## Метрики анализа



- **Глубина синтаксического дерева (tree_depth)** – максимальная длина цепочки зависимостей от корня предложения.  
- **Расстояние между зависимыми словами (dep_distance)** – среднее расстояние (в словах) между токеном и его синтаксической головой.  
- **Количество клауз (clauses)** – число предикативных единиц в предложении; при подсчете исключаются инфинитивы и деепричастия, не образующие самостоятельных предикативных единиц.  
- **Длина цепочки прилагательных (amod_chain)** – число прилагательных, зависящих от одного слова.  
- **Обстоятельственные придаточные (advcl)** – количество придаточных с обстоятельственным значением.  
- **Причастия (participles)** – число причастных оборотов (зависимость `acl`).  
- **Определительные придаточные (relcl)** – число придаточных, распространяющих имя (существительное, местоимение и др.) (`acl:relcl`).  
- **Деепричастия (verbal_adverbs)** – количество деепричастных оборотов (`advcl`).  
- **Изъяснительные придаточные (ccomp)** – число придаточных, дополняющих глагол.  
- **Инфинитивные обороты (infinitive_phrases)** – число инфинитивов, не входящих в составное сказуемое и не выступающих в роли подлежащего.  
- **Предложные определения (nmod)** – число существительных, зависящих от другого существительного и часто выступающих в роли определения.  

Каждый показатель вычисляется для каждого предложения, затем обобщается по всему тексту.
